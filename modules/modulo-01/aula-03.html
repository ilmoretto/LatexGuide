Curso de LaTeX

Aula 3: A Estrutura Mínima de um Documento

Todo documento LaTeX, não importa se tem uma ou mil páginas, obedece a uma estrutura fundamental. Nesta aula, vamos analisar o código mínimo necessário para criar um documento e entender o papel de cada comando.

1. Embasamento Teórico

Um arquivo .tex é dividido em duas partes lógicas: o Preâmbulo e o Corpo do Documento. O Preâmbulo é onde você define as configurações globais do documento: o tipo de documento, pacotes a serem usados, informações de título, etc. O Corpo é onde o conteúdo visível do documento (texto, figuras, tabelas) é de fato escrito.

2. Apresentação do Conteúdo

Analisaremos o código de um "Olá, Mundo!" em LaTeX, explicando linha por linha sua função.

3. Desenvolvimento: Anatomia do Código

O código abaixo é o mais simples que podemos ter para gerar um documento com algum conteúdo.

Código LaTeX

\documentclass{article}

\begin{document}

Olá, mundo! Este é o meu primeiro documento em LaTeX.

\end{document}

Visualização do Resultado

Olá, mundo! Este é o meu primeiro documento em LaTeX.

Desvendando cada comando:

  • \documentclass{article}

    Este é sempre o primeiro comando de qualquer documento LaTeX. Ele define a "classe" ou tipo de documento que estamos criando. A classe article é a mais comum, ideal para artigos, relatórios pequenos e outros textos curtos. Ela define formatações padrão para coisas como títulos, seções e tamanho da página. Outras classes importantes que veremos são report (para trabalhos mais longos, com capítulos) e book.

  • \begin{document}

    Este comando marca o fim do preâmbulo e o início do corpo do documento. Todo o conteúdo que deve aparecer no PDF final precisa estar depois deste comando.

  • Olá, mundo! ...

    Este é o conteúdo textual do nosso documento. O texto que você digita aqui é processado pelo LaTeX e inserido no PDF final, com formatação de parágrafo automática.

  • \end{document}

    Este comando sinaliza o fim do documento. O LaTeX ignora completamente qualquer coisa que venha depois desta linha. É crucial que todo \begin{document} tenha um \end{document} correspondente.

4. Atividade Prática

Vamos praticar no projeto "CursoLaTeX-Modulo1" que você criou no Overleaf.

  1. Apague todo o código que veio no template e substitua pelo código mínimo que vimos nesta aula.
  2. Altere a frase "Olá, mundo!..." por um parágrafo de 2 ou 3 frases sobre você.
  3. Clique em "Recompile" e observe como o PDF é atualizado com seu texto.
  4. Experimente: O que acontece se você esquecer de colocar o \end{document} no final? (Dica: o Overleaf provavelmente mostrará um erro). Corrija o erro e recompile.
