Curso de LaTeX

Aula 4: Estilizando o Texto — Fontes e Tamanhos

1. Embasamento Teórico

Embora a filosofia do LaTeX seja focar na estrutura, existem momentos em que é necessário dar ênfase a uma palavra ou trecho específico. O LaTeX oferece um conjunto robusto de comandos para alterar o estilo da fonte (família, peso e forma) e seu tamanho. É crucial entender a diferença entre comandos (que afetam apenas o texto dentro de seus argumentos) e declarações (que afetam todo o texto subsequente dentro de um escopo). Para o autoestudo, focaremos nos comandos por serem mais seguros e intuitivos.

2. Apresentação do Conteúdo

Nesta aula, você aprenderá os principais comandos para modificar a aparência do texto. Cobriremos a alteração da forma (itálico, inclinado), do peso (negrito), da família (serifa, sem serifa, monoespaçada) e do tamanho relativo da fonte.

3. Desenvolvimento do Conteúdo

Mudando a Forma e o Peso da Fonte

Estes são os comandos mais comuns para dar destaque ao texto. Eles recebem o texto a ser modificado como um argumento entre chaves {}.

Comandos de Formatação:

  • \textbf{texto em negrito} → texto em negrito
  • \textit{texto em itálico} → texto em itálico
  • \texttt{texto monoespaçado} → texto monoespaçado (ideal para código ou URLs)
  • \textsc{Texto em Versalete} → Texto em Versalete
  • \underline{texto sublinhado} → texto sublinhado

Aninhamento: \textbf{\textit{negrito e itálico}} → negrito e itálico

Mudando o Tamanho da Fonte

O LaTeX possui comandos para alterar o tamanho da fonte de forma relativa ao tamanho base. Para aplicar a um trecho específico, use: {\comandoDeTamanho texto}

Comandos de Tamanho (do menor ao maior):

  • {\tiny texto muito pequeno}
  • {\scriptsize texto bem pequeno}
  • {\footnotesize texto pequeno}
  • {\small texto um pouco menor}
  • {\normalsize texto normal}
  • {\large texto grande}
  • {\Large texto maior}
  • {\LARGE texto bem maior}
  • {\huge texto enorme}
  • {\Huge texto gigante}

Exemplo Prático Integrado

\documentclass[12pt, a4paper]{article}
\begin{document}

Este é um texto com tamanho \textbf{normal}.

Para dar ênfase, podemos usar o \textit{itálico}. 
Para código, usamos o tipo \texttt{monoespaçado}.

Podemos também controlar o tamanho:
{\small este trecho é um pouco menor.} 
Este trecho voltou ao normal.

E para grandes destaques:
{\LARGE CUIDADO!}

\end{document}

4. Atividade Prática

  1. No seu projeto Overleaf, escolha um dos parágrafos que você já escreveu.
  2. Coloque pelo menos uma palavra em negrito e outra em itálico.
  3. Imagine que você está citando o nome de um software ou um site. Coloque esse nome em fonte monoespaçada.
  4. Crie uma nova frase no final do parágrafo. Nela, faça a primeira palavra ser \large e a última \small. Lembre-se de usar as chaves {} para limitar o efeito dos comandos de tamanho.
  5. Recompile o documento e verifique se todas as formatações foram aplicadas corretamente. Observe como o texto após o trecho com tamanho modificado retorna automaticamente ao tamanho normal.
