1. A Era do UTF-8

Antigamente, trabalhar com acentos no LaTeX era um pesadelo. Hoje, com UTF-8, é simples e natural! Vamos aprender a configuração moderna que resolve todos os problemas de caracteres especiais.

Configuração Moderna Essencial

🎯 Template Base para Qualquer Documento

\documentclass[12pt]{article}

% Configuração moderna de encoding
\usepackage[utf8]{inputenc}     % Entrada UTF-8
\usepackage[T1]{fontenc}        % Saída de fontes
\usepackage[portuguese]{babel}  % Idioma português

\title{Texto com Acentos}
\author{Você}
\date{\today}

\begin{document}
\maketitle

Agora posso escrever naturalmente:
acentuação, coração, São Paulo, 
programação, não, João, ação!

\end{document}

Resultado: Todos os acentos aparecem perfeitamente no PDF final!

2. Os Três Pacotes Essenciais

inputenc - Entrada de Caracteres

📥 Como o LaTeX Interpreta o que Você Digita

Função: Diz ao LaTeX como interpretar os caracteres que você digita no editor

\usepackage[utf8]{inputenc}

Sem este pacote: ã vira ??, ç vira ??, é vira ??

fontenc - Saída de Caracteres

📤 Como os Caracteres Aparecem no PDF

Função: Controla como os caracteres são exibidos na saída final

\usepackage[T1]{fontenc}

Vantagem: Permite copiar e buscar texto com acentos no PDF gerado

babel - Suporte a Idiomas

🌍 Adaptação às Regras do Idioma

Função: Adapta o LaTeX às regras tipográficas do idioma escolhido

\usepackage[portuguese]{babel}  % ou [brazil]
\usepackage[english]{babel}     % Inglês
\usepackage[spanish]{babel}     % Espanhol
\usepackage[french]{babel}      % Francês

Benefícios: Hifenização correta, datas em português, "Capítulo" ao invés de "Chapter"

4. Comandos de Escape (Método Antigo)

Ainda é útil conhecer os comandos tradicionais para casos especiais ou quando UTF-8 não estiver disponível.

6. Resolvendo Problemas Comuns

Principais Erros e Soluções

Erro: "Package inputenc Error"
Problema: Arquivo não está em UTF-8
Solução: No seu editor, salve o arquivo como UTF-8

Problema: Acentos aparecem estranhos no PDF
Causa: Falta fontenc
Solução: Adicione \usepackage[T1]{fontenc}

Problema: "Chapter" ao invés de "Capítulo"
Causa: Falta babel
Solução: Adicione \usepackage[portuguese]{babel}

Template de Segurança

💡 Sempre Comece Assim

% Template base para qualquer documento:
\documentclass{article}
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage[portuguese]{babel}

7. Prática: Documento Multilíngue

Vamos criar um documento com diferentes idiomas e caracteres especiais para consolidar o aprendizado.

🌍 Exemplo Prático Completo

\documentclass[12pt,a4paper]{article}

% Configuração completa
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage[portuguese,english,spanish,french]{babel}

\title{Documento Internacional}
\author{Autor Multilíngue}
\date{\today}

\begin{document}
\maketitle

\section{Português}
\selectlanguage{portuguese}
Este é um texto em português com acentuação correta:
programação, coração, não, ação, João.

A temperatura em São Paulo é 25°C.
O preço é R\$ 10,50.

\section{English}
\selectlanguage{english}
This is a text in English. Notice how the section
title automatically changed to "Section" instead
of "Seção".

\section{Español}
\selectlanguage{spanish}
Este es un texto en español con acentos:
programación, corazón, niño, año.

\section{Français}
\selectlanguage{french}
Ceci est un texte en français avec des accents:
café, hôtel, naïf, Noël.

\end{document}

Observe: Cada seção muda automaticamente os termos para o idioma correto. "Section" em inglês, "Sección" em espanhol, "Section" em francês!
