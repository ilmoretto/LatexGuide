📖 A Era do UTF-8

Antigamente, trabalhar com acentos no LaTeX era um pesadelo. Hoje, com UTF-8, é simples e natural!

🎯 Configuração Moderna Essencial

\documentclass[12pt]{article}

% Configuração moderna de encoding
\usepackage[utf8]{inputenc}     % Entrada UTF-8
\usepackage[T1]{fontenc}        % Saída de fontes
\usepackage[portuguese]{babel}  % Idioma português

\title{Texto com Acentos}
\author{Você}
\date{\today}

\begin{document}
\maketitle

Agora posso escrever naturalmente:
acentuação, coração, São Paulo, 
programação, não, João, ação!

\end{document}

💡 Resultado: Todos os acentos aparecem perfeitamente!

🔤 Os Três Pacotes Essenciais

📥 inputenc

O que faz: Diz ao LaTeX como interpretar os caracteres que você digita

\usepackage[utf8]{inputenc}

Sem isso: ã vira ??, ç vira ??, é vira ??

📤 fontenc

O que faz: Controla como os caracteres são exibidos na saída

\usepackage[T1]{fontenc}

Vantagem: Permite copiar e buscar texto com acentos no PDF

🌍 babel

O que faz: Adapta o LaTeX às regras do idioma escolhido

\usepackage[portuguese]{babel}  % ou [brazil]
\usepackage[english]{babel}     % Inglês
\usepackage[spanish]{babel}     % Espanhol
\usepackage[french]{babel}      % Francês

Benefícios: Hifenização correta, datas em português, "Capítulo" ao invés de "Chapter"

🔧 Comandos de Escape (Método Antigo)

Ainda é útil conhecer os comandos tradicionais para casos especiais:

Caractere Comando LaTeX Exemplo
á \'{a} a\c{c}\'ao
à \`{a} \`aquele
â \^{a} voc\^e
ã \~{a} n\~ao
ç \c{c} Fran\c{c}a
ü \"{u} Br\"uke

💡 Dica: Use os comandos de escape apenas quando UTF-8 não estiver disponível ou para caracteres muito raros!

🔍 Resolvendo Problemas Comuns

⚠️ Erro: "Package inputenc Error"

Problema: Arquivo não está em UTF-8

Solução: No seu editor, salve o arquivo como UTF-8

⚠️ Acentos aparecem estranhos no PDF

Problema: Falta fontenc

Solução: Adicione \usepackage[T1]{fontenc}

⚠️ "Chapter" ao invés de "Capítulo"

Problema: Falta babel

Solução: Adicione \usepackage[portuguese]{babel}

💡 Template de Segurança

% Sempre comece seus documentos assim:
\documentclass{article}
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage[portuguese]{babel}

🎯 Prática: Documento Multilíngue

Vamos criar um documento com diferentes idiomas e caracteres especiais:

\documentclass[12pt,a4paper]{article}

% Configuração completa
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage[portuguese,english,spanish,french]{babel}

\title{Documento Internacional}
\author{Autor Multilíngue}
\date{\today}

\begin{document}
\maketitle

\section{Português}
\selectlanguage{portuguese}
Este é um texto em português com acentuação correta:
programação, coração, não, ação, João.

A temperatura em São Paulo é 25°C.
O preço é R\$ 10,50.

\section{English}
\selectlanguage{english}
This is a text in English. Notice how the section
title automatically changed to "Section" instead
of "Seção".

\section{Español}
\selectlanguage{spanish}
Este es un texto en español con acentos:
programación, corazón, niño, año.

\section{Français}
\selectlanguage{french}
Ceci est un texte en français avec des accents:
français, café, hôtel, naïve.

\section{Símbolos Especiais}
\selectlanguage{portuguese}
% Volta para português

Símbolos matemáticos: ±, ×, ÷, °, ≤, ≥
Moedas: €, $, £, ¥
Outros: ©, ®, ™, §, ¶

\end{document}

🎯 Exercício: Compile este documento e observe como:

  • Todos os acentos aparecem corretamente
  • As palavras "Section", "Seção", "Sección" mudam automaticamente
  • A hifenização respeita as regras de cada idioma
  • Os símbolos especiais funcionam perfeitamente
